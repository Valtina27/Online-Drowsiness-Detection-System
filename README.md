# ONLINE DROWSINESS DETECTION SYSTEM

## REAL-TIME DRIVER DROWSINESS DETECTION USING COMPUTER VISION AND MACHINE LEARNING

---

## OVERVIEW

The Online Drowsiness Detection System is an Artificial Intelligence-based driver safety application developed as a Phase I Capstone Project for the Bachelor of Science in Artificial Intelligence and Data Science program.

The system is designed to monitor a driver's facial movements in real time using Computer Vision and Machine Learning techniques. By continuously analyzing eye closure, blinking patterns, and yawning behavior, the application detects signs of driver fatigue and immediately generates alerts to reduce the risk of road accidents caused by drowsiness.

The primary objective of this project is to improve road safety through early detection of driver fatigue using intelligent vision-based technologies.

---

## SYSTEM WORKFLOW

1. Capture live video using the webcam.
2. Detect the driver's face.
3. Identify facial landmarks.
4. Calculate the Eye Aspect Ratio (EAR).
5. Detect prolonged eye closure and yawning.
6. Analyze facial behavior using Machine Learning.
7. Generate an alert if drowsiness is detected.
8. Continue monitoring in real time.
